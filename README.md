# WRT — Work, Relax, Talk

Modern school social network built for communication, collaboration and friendly competition between classes.

## ✨ Features

- User profiles
- Posts and comments
- Real-time chat
- Notifications
- Class Clash
- Achievements & XP
- Analytics Dashboard
- Moderation System

## 🏆 Class Clash

Class Clash is a gamified competition where classes earn points through activity and compete for the top position on the leaderboard.

## 🛠️ Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS

### Backend
- Django
- Django REST Framework
- PostgreSQL
